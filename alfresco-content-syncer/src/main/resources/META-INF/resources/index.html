Quarkus UI • local binaries to Alfresco

Alfresco Content Syncer

Minimal operational UI to launch local folder synchronization jobs, track status, and inspect the final report. The sync runs on the same host where this application is started.

Operational Notes

Use dry run first. Remote delete is disabled by default because it is destructive. Existing files are updated only when size differs or the local timestamp is newer than Alfresco.
